## Packages
framer-motion | Complex animations and page transitions for the luxury feel
lucide-react | Beautiful, consistent icons
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility to merge Tailwind classes without conflicts
embla-carousel-react | For the product image carousel
react-hook-form | Form state management
zod | Schema validation
@hookform/resolvers | Zod resolver for react-hook-form

## Notes
- Images should use high-quality Unsplash URLs as placeholders
- Design system uses 'Playfair Display' for headings and 'DM Sans' for body
- Auth is handled via Replit Auth (use-auth hook)
- Cart is server-side persisted via /api/cart
